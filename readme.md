# Emotion Cipher 🔐

Emotion Cipher is an AI-powered web application that encrypts text messages while preserving their emotional meaning using NLP-based emotion detection.

---

## 🚀 Features

* Emotion detection using NLP
* Secure message encryption
* Decryption with emotion recovery
* Emotion score visualization
* Interactive Streamlit interface

---

## 🛠 Tech Stack

* Python
* Streamlit
* HuggingFace Transformers
* Cryptography
* Pandas

---

## ▶️ Run Locally

```bash
pip install -r requirements.txt
streamlit run app.py
```

## 📌 Project Idea

Emotion Cipher explores the concept of **Emotion-Aware Encryption**, where the message text remains encrypted for privacy while the emotional meaning can still be detected by AI systems.
