# Emotion Cipher 🔐

Emotion Cipher is a web-based AI system that encrypts text messages while preserving their emotional meaning.

## Features
- Emotion detection using NLP
- Secure message encryption
- Decryption with emotion recovery
- Emotion score visualization
- Interactive Streamlit interface

## Tech Stack
Python
Streamlit
HuggingFace Transformers
Cryptography
Pandas

## Run Locally
pip install -r requirements.txt
streamlit run app.py
